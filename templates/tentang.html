Apa itu MediExpert?

MediExpert adalah sistem pakar berbasis web yang dirancang untuk membantu pengguna mengidentifikasi kemungkinan penyakit berdasarkan gejala yang mereka alami. Sistem ini menggunakan metode Forward Chaining — salah satu teknik inferensi utama dalam kecerdasan buatan — untuk mencocokkan gejala dengan basis aturan yang telah dikurasi oleh pakar medis.

Aplikasi ini dibangun menggunakan Python Flask sebagai backend dan HTML5/CSS3/JavaScript modern sebagai frontend, dengan penyimpanan riwayat konsultasi menggunakan SQLite.

Metode Forward Chaining

Working Memory

Gejala yang dipilih pengguna dimasukkan sebagai fakta awal ke dalam memori kerja.

facts = {"G01", "G03", "G05"}

Rule Matching

Mesin inferensi mengevaluasi setiap rule: apakah kondisi IF terpenuhi dalam working memory?

IF {G01, G05, G06} ⊆ facts
→ THEN Influenza

Scoring

Skor kecocokan dihitung: 60% dari gejala wajib + 40% dari gejala pendukung.

skor = 60% + (pendukung/total × 40%)

Kesimpulan

Penyakit dengan skor ≥70% menjadi diagnosa utama. Skor 40–70% jadi kemungkinan.

≥70% → Diagnosa Utama
40-70% → Kemungkinan
